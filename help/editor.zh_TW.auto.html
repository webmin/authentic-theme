編輯器鍵盤快捷鍵
查找/替換
按鍵命令
Ctrl + F尋找
Ctrl + + F代替
Ctrl + G找下一個
Ctrl + + G查找上一個
Alt + G跳到行
文本操作
按鍵命令
Alt + W切換軟包裝
Ctrl + KU轉換為大寫
Ctrl + KL轉換為小寫
編輯
按鍵命令
Esc, Esc關閉活動編輯器窗口
Ctrl + Esc, Esc最小化活動編輯器窗口
Alt + 最大化活動編輯器窗口
Alt + 規範化活動編輯器窗口
Ctrl + S節省
Ctrl + + S另存為..
Ctrl + Alt + + S保存並關閉
Alt + S在文件中打開搜索
Alt + Space運行自動完成
Ctrl + X切割線
Ctrl + + 向上移動行/選擇
Ctrl + + 向下移動行/選擇
Ctrl + L選擇行 - 重複以選擇下一行
Ctrl + D選擇單詞 - 重複選擇其他出現
Ctrl + M跳到右括號 - 重複跳到左括號
Ctrl + + M選擇當前括號的所有內容
Ctrl + + K刪除線
Ctrl + KK從光標處刪除到行尾
Ctrl + K + Backspace從光標刪除到行首
Ctrl + + D重複行
Ctrl + J將下面的行加入到當前行的末尾
Ctrl + /評論/取消評論
Ctrl + Y重做或重複上一個鍵盤快捷鍵命令
Ctrl + + V正確粘貼和縮進
Ctrl + U軟撤消；重複時在撤消更改之前跳轉到您的最後一個更改
