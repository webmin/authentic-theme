Клавишни комбинации на редактора
Намиране/замяна
Натискане на клавиш командване
Ctrl + F намирам
Ctrl + + F Сменете
Ctrl + G Намери следващото
Ctrl + + G Намери предишен
Alt + G Преминете към ред
Текстова манипулация
Натискане на клавиш командване
Alt + W Превключване на мека обвивка
Ctrl + KU Преобразуване в главни букви
Ctrl + KL Преобразувайте в малки букви
Редактиране
Натискане на клавиш командване
Esc, Esc Затворете активния прозорец на редактора
Ctrl + Esc, Esc Минимизиране на активния прозорец на редактора
Alt + Увеличете активния прозорец на редактора
Alt + Нормализиране на активния прозорец на редактора
Ctrl + S Запазване
Ctrl + + S Запази като..
Ctrl + Alt + + S Запази и затвори
Alt + S Отворете търсене във файлове
Alt + Space Стартирайте автоматично довършване
Ctrl + X Линия на изрязване
Ctrl + + Преместване на ред/селекция нагоре
Ctrl + + Преместване на линия/селекция надолу
Ctrl + L Изберете ред - Повторете, за да изберете следващите редове
Ctrl + D Изберете дума - Повторете избора на други срещания
Ctrl + M Преминаване към затварящи скоби - Повторете, за да преминете към отварящи скоби
Ctrl + + M Изберете цялото съдържание на текущите скоби
Ctrl + + K Изтриване на линия
Ctrl + KK Изтриване от курсора до края на реда
Ctrl + K + Backspace Изтриване от курсора до началото на реда
Ctrl + + D Дублирани редове
Ctrl + J Присъединете реда по-долу към края на текущия ред
Ctrl + / Коментиране/прекоментиране
Ctrl + Y Повторете или повторете последната команда за клавишна комбинация
Ctrl + + V Поставете и отстъпете правилно
Ctrl + U Меко отмяна; прескача до последната ви промяна, преди да отмени промяната, когато се повтори
