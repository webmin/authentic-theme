Atalhos de teclado do editor
Localizar/Substituir
Pressione o botão Comando
Ctrl + F Achar
Ctrl + + F Substituir
Ctrl + G Encontre o próximo
Ctrl + + G Encontrar anterior
Alt + G Ir para a linha
Manipulação de texto
Pressione o botão Comando
Alt + W Alternar envoltório macio
Ctrl + KU Transformar para maiúscula
Ctrl + KL Transformar em minúsculas
Editando
Pressione o botão Comando
Esc, Esc Fechar a janela do editor ativo
Ctrl + Esc, Esc Minimizar a janela do editor ativo
Alt + Maximizar a janela do editor ativo
Alt + Normalizar a janela do editor ativo
Ctrl + S Salvar
Ctrl + + S Salvar como..
Ctrl + Alt + + S Salvar e fechar
Alt + S Abrir pesquisa em arquivos
Alt + Space Executar preenchimento automático
Ctrl + X Linha de corte
Ctrl + + Mover linha/seleção para cima
Ctrl + + Mover linha/seleção para baixo
Ctrl + L Selecionar linha - Repita para selecionar as próximas linhas
Ctrl + D Selecionar palavra - Repetir selecionar outras ocorrências
Ctrl + M Pular para fechar parênteses - Repita para pular para abrir parênteses
Ctrl + + M Selecione todo o conteúdo dos parênteses atuais
Ctrl + + K Excluir linha
Ctrl + KK Excluir do cursor até o final da linha
Ctrl + K + Backspace Excluir do cursor para o início da linha
Ctrl + + D Linha(s) duplicada(s)
Ctrl + J Unir a linha abaixo ao final da linha atual
Ctrl + / Comentar/descomentar
Ctrl + Y Refazer ou repetir o último comando de atalho de teclado
Ctrl + + V Colar e recuar corretamente
Ctrl + U Desfazer suave; salta para sua última alteração antes de desfazer a alteração quando repetida
